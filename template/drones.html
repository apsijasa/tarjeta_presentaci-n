El Proyecto

Nuestro proyecto busca desarrollar un drone funcional utilizando tecnología de impresión 3D, combinando diseño moderno, componentes electrónicos de alta calidad e inteligencia artificial para crear una solución versátil de vigilancia y monitoreo.

Objetivos Principales:

  • Fabricar un drone funcional utilizando impresión 3D
  • Presentar el primer diseño multifuncional
  • Conocer la electrónica del drone y componentes
  • Integración de IA para reconocimiento básico
  • Determinar costos y tiempos de construcción
  • Establecer la cadena de suministros de componentes electrónicos
  • Identificar obstáculos de fabricación
  • Crear un modelo de negocio viable

Características y Beneficios

Fabricación con Impresión 3D

Utilizamos tecnología de impresión 3D para crear componentes personalizados y de bajo peso, permitiendo un diseño modular y fácil mantenimiento.

Larga Duración de Vuelo

Diseñado para vuelos prolongados con una duración de batería que supera las 4 horas, ideal para misiones de vigilancia y monitoreo continuo.

Integración con IA

Incorporamos sistemas de inteligencia artificial para reconocimiento de objetos y análisis de datos en tiempo real.

Fácil Mantenimiento

Componentes modulares que permiten una sustitución rápida y sencilla, reduciendo el tiempo de inactividad.

Sensores Múltiples

Capacidad para incorporar diferentes tipos de sensores según las necesidades específicas de cada misión.

Diseño Aerodinámico

Estructura optimizada para eficiencia aerodinámica, basada en análisis CFD y pruebas de vuelo reales.

Fases de Desarrollo

Fase 1: Diseño

Concepto: Desarrollo del concepto del drone funcional para largas horas de vuelo con capacidad para diferentes sensores.

Aerodinámica: Análisis y optimización aerodinámica mediante CFD.

Diseño 3D: Creación del diseño completo para su posterior impresión.

Fase 2: Impresión e Implementación Técnica

Impresión: Fabricación de todas las piezas mediante impresión 3D.

Electrónica: Instalación de componentes electrónicos y configuración.

Software: Desarrollo e implementación del software de control y IA.

Fase 3: Capacitación del Personal

Pilotos: Formación de pilotos para manejo del drone.

Analistas: Capacitación para interpretación de datos aéreos.

Protocolos: Implementación de protocolos de actuación y vuelo.
